# 修改 macOS 系统网络DNS设置

1. 打开系统偏好设置

2. 点击“网络”
![系统偏好设置][system-preferences]

3. 左侧选中当前连接的网卡，然后点击右下角“高级”按钮
![网络设置][network]

4. 选择“DNS”分栏

5. 单击左下方的“+”添加按钮

6. 输入添加 `223.5.5.5` 和 `223.6.6.6`
![填写DNS][fill-dns]

7. 点击右下角“好”，会回到网络面板，再点击一次“应用”

8. 完毕
